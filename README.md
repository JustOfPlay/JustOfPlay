# 💫 About Me:
Hi, I'm Luis, a small developer from Germany. I'm here to improve my programming skills and learn more about the tools and technologies around software development.
I'm currently learning Rust and already have experience with Java. I'm especially interested in systems programming, low-level concepts, and building efficient and reliable software.
I use Arch Linux as my daily driver and Neovim as my main editor. I enjoy working in a minimal and flexible environment and like to understand my whole setup from the ground up instead of relying on prebuilt abstractions.
Most of my time goes into coding, experimenting with Linux, and exploring different areas of software development like automation and self-hosting. I mainly learn by building projects and figuring things out along the way.
